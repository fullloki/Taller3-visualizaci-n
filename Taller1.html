Introducción

A continuación se muestran conceptos básicos para la creación de una estructura web en HTMl. Conoceremos sus sintaxis básica para implementarla en un documento.

¿Qué es HTML?

Veamos primero la definición de HTMl. Las siglas HTML significa Hyper Text Markup Language o en español Lenguaje de Marcado de Hipertexto. Este consiste en una series de elementos que le dice a los navegadores como va a mostrarse el contenido en nuestra Web.

Ejemplo de un documento simple en HTML

            
                <!DOCTYPE html>
                <html>
                <head>
                    <title> Mi página </title>
                </head>
                <body>
                    <h1> Encabezado </h1>
                    <p> parrafp </p>
                </body>
                </html>

Elementos básicos de un documento HTML:

  1. <!DOCTYPE html> define el documento como uno HTML
  2. <html> abre la estructura del HTML
  3. <head>es la cabecera del HTML. Contiene información sobre la página
  4. <title> Muestra en la barra del navegador el titulo de la página.
  5. <body>cuerpo del documento HTML. Aquí colocamos todo el contenido que queremos en nuestra página
  6. <h>define los encabezados
  7. <p>define un parrafo en el documento

Tablas

Las tablas HTML permiten a los desarrolladores web organizar los datos en filas y columnas. Para crear una tabla en HTML utilizamos <table>, para las filas <tr> y para las columnas <td>.
Ejemplo de codigo: tabla de 1 filas y 2 columnas 

            
                <table>
                    <tr>
                        <td>
                            fila 1, columna 1
                        </td>
                        <td>
                            fila 1, columna 2
                        </td>
                    </tr>
                </table>

Ejemplo de como se vería

fila 1, columna 1 fila1, columna 2

Ejemplo de codigo:

                        </li>

Atibutos HTML

Los atributos proveen informacion adicional sobre un elemento o estiqueta, a la vez
que lo modifica. La mayoria de los atributos tienen un "valor"; el valor modifica el atributo.

  • Los atributos siempre se especifican en la etiqueta de inicio
  • Los atributos generalmente vienen en pares de nombre/valor como: nombre="valor"

    Todos los elementos html pueden tener atributos

  • El href atributo de *a* especifica la URL de la pagina a la que va el enlace
  • El src atributo de *img* especifica la ruta a la imagen que se mostrará
  • Los atributos width y height de *img* proporcionan informacion sobre el tamaño de la imagen
  • El alt atributo de *img* proporciona un texto alternativo para una imagen
  • El style atributo se usa para agregar estilos a un elemeto, como color, fuente, tamaño y mas
  • El lang atributo de *html* declara el idioma de la pagina web
  • El title atributo define alguna informacion adicional sobre un elemento

Colores

Hay una gran variedad de combinaciones de colores que puedes aplicar en HTML.

Podemos utilizar el atributo style para darle estilo a un elemento. En este caso para darle color a un parrafo podemos utilizar:

  • <p style="color:red;"> : Formato nombre del color
  • <p style="color:#ff0000;"> : Formato Hexadecimal
  • <p style="color:rgb(255, 0, 0);"> : Formato RGB

Este es un parrafo en color rojo.

Atributo de href

Los hipervinculos son definidos usando la etiqueta <a>.Use el atributo href="" para definir la direccion de destino del Hipervinculo.
Ejemplo: <a href="aqui va la direccion"> texto para ir al link</a>... Probando la direccion

Atributo src con contenido multimedia

Imagen

La etiqueta <img> es usada para insertar una imagen. No contiene una etiqueta de cierre. La URL de la imagen (direccion) puede ser definida usando el atributo src="". Ejemplo:
<img src="image.jpg"/>...
Se puede modificar la altura de la imagen con el atributo height;.
Se puede modificar la anchura de la imagen con el atributo width.
Se utilizan de la siguiente manera <img src="image.jpg" width="125" height="100">
El atributo alt="" especifica un texto alternativo que describe la imagen.

Audio

La etiqueta <audio> es usada para reproducir un archivo de audio en HTML. Podemos agregar los controles de reproducción, pausa y sonido con la etiqueta <controls>. Ejemplo: 

                    
                        <audio src="./Mcpe_xp_ sound.mp3" controls </audio>

Video

La etiqueta <video> es usada para reproducir un archivo de video en HTML. Podemos agregar los controles de reproducción, pausa y sonido con el atributo controls.Ejemplo: 

                    
                        <video  src="ruta de video" controls></video>

Formularios

Para crear un formulario en HTML se utiliza la etiqueta <form> y dentro de esta podemos agregar los siguientes elementos:

tipo descripción
<label for=""> Muestra una etiqueta de información
<input type="text"> Muestra un campo de entrada de texto de una sola línea
<input type="radio"> Muestra un botón de radio (para seleccionar una de las muchas opciones)
<input type="checkbox"> Muestra una casilla de verificación (para seleccionar cero o más de muchas opciones)
<input type="submit"> Muestra un botón de envío (para enviar el formulario)
<input type="button"> Muestra un botón en el que se puede hacer clic
